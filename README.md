## About This Portfolio

This repository contains AI-powered automation workflows built with n8n and Make.com. Each project solves a real business problem using free tools and APIs, demonstrating practical skills in workflow automation, API integration, and AI implementation.

## Projects

### 1. Sales Lead Automation Pipeline
An end-to-end lead management system that captures leads from a Google Form, scores them using Google Gemini AI, creates contacts in HubSpot CRM automatically, logs everything to Google Sheets, and sends Gmail notifications — all in under 10 seconds without any manual work.

Tools used: n8n, Google Forms, Google Gemini API, HubSpot CRM, Google Sheets, Gmail

[View Project](./sales-lead-pipeline)

### 2. Invoice Processing Bot
An automated invoice management system that watches Gmail for invoice emails, uses Google Gemini AI to extract vendor, amount, date and category, logs everything to both Notion and Google Sheets, and sends budget alert emails for invoices above a set threshold.

Tools used: Make.com, Gmail, Google Gemini API, Notion, Google Sheets

[View Project](./invoice-processing-bot)
